About the GA Engine

How the Genetic Algorithm Works

Each possible recommendation list is treated as a chromosome, where each product ID is a gene. The algorithm evolves a population of these chromosomes over many generations to find the best product set for each user.

  1. Initial Population - random product sets
  2. Fitness Function - scored from user views, clicks, purchases, and ratings
  3. Tournament Selection - best individuals are chosen as parents
  4. One-Point Crossover - parent chromosomes combine to form new child
  5. Swap Mutation - two genes randomly swap to keep diversity
  6. Elitism - top chromosomes pass directly to next generation
